Upstream forkladığınız git reposunun orjinal dosyalarını güncel tutmanıza yarar.

Komut satırı şu sekildedir;

```
git remote -v 

git remote add upstream ORJİNAL REPO URL'Sİ

git remote -v 

git fetch upstream

git merge upstream/master

git push origin master
```
